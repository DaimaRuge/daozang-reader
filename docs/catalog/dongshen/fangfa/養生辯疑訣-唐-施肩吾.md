# 養生辯疑訣-唐-施肩吾

---


### 養生辨疑訣

### 經名：養生辨疑訣。唐·施肩吾撰。一卷。底本出處：《正統道藏》洞神部方法類，參校版本：《雲笈七籤》，收入該書卷八十八。

### 養生辨疑訣

栖真子施肩吾述

一氣無方，與時消息，萬物生死，共氣盛衰。處自然之問，而皆不知所以然而然。其所稟習，在覆載之下。有形者先<sup>[1]</sup>知其本，知其本則求<sup>[2]</sup>無不通；脩道先須正其源，正其源則流無不應。若棄其本而外求，背其源以邪究，雖獵盡百家，學窮諸子，徒廣虛論功條，其攝養之效<sup>[3]</sup>，得者觀之，實為自誤<sup>[4]</sup>耳！
今歷觀世問，好道之流，不可勝數。雖知恬淡以自守，全不知恬淡之中有妙用矣；雖知虛無以為理，全不知虛無之中而無不為矣。若不知虛無恬淡妙用之理，徒委志於寂默之間，妄作於形神之外，是謂無益之用，非攝生之鴻漸也。

且神由形住，形以神留，神苟外遷，形亦難保。抑又服餌草木金石，以固其形，而不知草木金石之性，不究四時逆順之宜，久而服之，反傷和氣。遠不出中年之內，疾害俱生。使夫輕薄之流，皆謂繫風捕影，不可得矣。翻以學者為不肖，以真隱為詭道，不亦傷哉！

惑<sup>[5]</sup>人嘗以此事而譏余，曰：吾聞學道，可致長生。吾自童年至於暮齒，見學道之人已千數矣。服氣絕粒者，驅役考召者，清靜無欲者，修仙煉行者，如斯之流，未有不聞其死者也。身歿幽壤之下，徒以尸解為名。

推此而論之，至皿得者猶靈骨耳，非可學而得之。余聞斯論，不覺心愍<sup>[6]</sup>于內，神恍<sup>[7]</sup>于外。沉昤之問，乃太息而應之曰：觀子向來所說，實亦鄙之甚矣！迷之尤矣！今世人學人問<sup>[8]</sup>之事，猶有成與不成，又昆妙本玄深，昏昏默默，胡可造次而得之？

且大道無親，感之即應，苟雲靈骨，無乃疏乎！然夫服氣絕粒者，且道家之所尚，人苟得之，皆有不食之功，身輕之效<sup>[9]</sup>，便自言腸胃無滓，立致雲霓，形體獲輕，坐希鸞鶴。採餌者，復以毛女為憑；呼吸者，又引靈龜作證。

曾不知真氣暗滅<sup>[10]</sup>，胎精內枯，猶執滯理於松筠，守迷端於翰墨，良可嗟矣！寧不<sup>[11]</sup>怪乎？至於驅役考召之流，至皿是道中之法事。研討至精，窮其真誥，誠為身外之虛名，妄作人問之孟浪<sup>[12]</sup>。

在己無徵於延益，於人有驗於斡攘。亂構休祥，徒陳禍福。如斯之輩，並匪保生之道也。或以清靜無為，深居絕俗，形同槁木，志類死灰，不知天地動用之心，不察陰陽運行之理。如此則雖游恍惚，其恍惚而無涯；縱合育冥，其官冥而莫測。

翻使希夷之外，神用罔然；虛白之中，玄關失守。言議之際，中有高真。喟然而嘆曰：守一非一，履真非真。此亦近為門階之由，殊未窺其室中之用矣。大凡保氣棲神，不可以湛然而得之，亦不可以元<sup>[13]</sup>然而守之。

且神無方而氣常運，形至靜而用無窮。是知保氣者，其要在乎運；棲神者，其秘在乎用。吾嘗聞之於師曰：體虛而氣周，形靜而神會。此蓋為出世之玄機，無名之大用矣。

### 養生辨疑訣竟

<sup>[1]</sup>先：此下《雲笈七籤》有『須』字。

<sup>[2]</sup>求：《雲笈七籤》作『未』。

<sup>[3]</sup>徒廣虛論功條，其攝養之效：《雲笈七籤》作『徒廣虛論之功，終無攝養之效』。．<sup>[4]</sup>誤：《雲笈七籤》作『悟』。
<sup>[5]</sup>惑：咋雲笈七籤》作『或』。
<sup>[6]</sup>愍：此下《雲笈七籤》有『然』字。
<sup>[7]</sup>恍：此下《雲笈七籤》有『惚』字。
<sup>[8]</sup>人問：《雲笈七籤》作『凡問』。
<sup>[9]</sup>皆有不食之功，身輕之效：《雲笈七籤》作『皆有不食輕舉之效』。
<sup>[10]</sup>減：《雲望七籤》作『減』。
<sup>[11]</sup>不：原作『所』，據《雲笈七籤》改。
<sup>[12]</sup>妄作人問之孟浪：此句以下《雲笈七籤》無，另有異文四百八十七字。
<sup>[13]</sup>元：疑為『兀』之誤。
