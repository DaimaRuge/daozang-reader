# 洞真部

<div class="intro">

**洞真部**为三洞之首，主修存思服气，以上清经系为核心。本部共收录 **319** 部经典。

上清派兴起于东晋，以《上清大洞真经》为本，主修存思日月星斗、五脏神明，兼习服气导引、诵经斋戒。

</div>

## 🎯 推荐阅读

### 入门经典

1. **黃帝陰符經** - 文简义深，道家哲学精要
2. **上清大洞真經** - 上清派核心经典
3. **靈寶無量度人經** - 道教第一大经

### 修炼要籍

1. **大丹直指** - 全真北宗内丹心法
2. **悟真篇** - 内丹南宗祖经
3. **金丹大成集** - 金丹法诀集成

### 历史传记

1. **列仙傳** - 最早神仙传记
2. **历世真仙體道通鑒** - 大型人物通史
3. **漢武帝內傳** - 西王母传道故事
