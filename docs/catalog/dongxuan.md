# 洞玄部

<div class="intro">

**洞玄部**为三洞之二，主修斋醮科仪，以灵宝经系为核心。本部共收录 **303** 部经典。

灵宝派起源于东晋末，以《灵宝度人经》为本，重视斋醮济度、符箓咒术，对后世道教影响深远。

</div>
